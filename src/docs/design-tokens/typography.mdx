# Typography

This section outlines the typography guidelines and standards for our design system.
The available typography uses [Poppins](https://fonts.google.com/specimen/Poppins?query=Poppins) as the primary font
and `Arial*` as its fallback.

`Poppins` is imported in the source files and therefore also available in the packages out of the box.

Every font can be selected with `mb-font-*`

## Tokens

- `mb-font-h1`
- `mb-font-h2`
- `mb-font-h3`
- `mb-font-h4`
- `mb-font-paragraph-l`
- `mb-font-paragraph-m`
- `mb-font-label-xl`
- `mb-font-label-l`
- `mb-font-label-m`
- `mb-font-label-s`
- `mb-font-placeholder`
